## Weather Information Tool

A Python CLI application that retrieves real-time weather data for a user-specified city using the OpenWeatherMap API.

### Features
- Live weather data via REST API
- Temperature, humidity, wind speed, and conditions
- Secure API key handling using environment variables
- Error handling for invalid locations and API failures
